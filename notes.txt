Start with playlist ID and BPM as inputs

-Login using spotify auth
-Get all user playlists, select one
-Get all spotify IDs from playlist tracks
-Iterate IDs to find songs that have tempos that are factors/multiples of bpm
-Output tracks matching search
-Option to add playlist to library

Retreive BPM (tempo field in response json):
https://developer.spotify.com/documentation/web-api/reference/get-audio-features

Spotify branding guidelines:
https://developer.spotify.com/documentation/design

Logo usage:
https://developer.spotify.com/documentation/design#using-our-logo

Using the following tutorial:
https://www.newline.co/courses/build-a-spotify-connected-app

To run app:
    Run [npm_start] in terminal and visit the url it says
    See package.json "scripts" to see that it starts the server
    Use on incognito to avoid login and auth issues
